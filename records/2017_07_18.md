# Anomaly Detection By RNN

撰寫 Encoder、Decoder 預測 Anomaly

## Usage

``` bash
/src $ python anomaly-detection.py \
    --scope phm2012 \
    --name ad-phm \
    --step-size 16 \
    --input-size 2 \
    --hidden-size 16 \
    --output-size 2 \
    --batch-size 512 \
    --layer-depth 2 \
    --dropout-rate 0.05 \
    --learning-rates \
        1  1000   0.001 \
    --sample-size 100
```
